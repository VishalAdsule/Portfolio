Projects

Things I've Built

June 2025

Garage Billing System

An automated billing system that streamlines real-time invoicing and payment processing for a garage, with centralized records to manage customers, vehicles, and service history.

  • Real-time job tracking and inventory management
  • Centralized database for customer & vehicle service records
  • Complete digital service history per vehicle
Java JDBC IntelliJ IDEA MySQL
GitHub

Problem: Garage owners were tracking invoices, inventory, and vehicle history manually, leading to billing errors and lost records.

Solution: A Java + JDBC desktop application connected to a centralized database, automating invoicing and consolidating customer/vehicle data.

Challenges: Designing a relational schema that cleanly linked customers, vehicles, jobs, and inventory while keeping queries fast.

Outcome: Reduced manual billing errors and gave the garage a searchable, complete service history for every vehicle.

September 2024

Stock Market Prediction Using ML

A deep learning model using RNN and LSTM networks for real-time stock market forecasting, backed by an automated data pipeline for live market data.

  • RNN & LSTM networks for time-series forecasting
  • Automated pipeline to acquire, clean, and pre-process live API data
  • Performance benchmarked against traditional statistical models
Python LSTM / RNN VS Code
GitHub

Problem: Traditional statistical models struggled to capture the non-linear patterns in live stock market data.

Solution: Built an LSTM/RNN deep learning pipeline that ingests live API data, pre-processes it automatically, and forecasts trends.

Challenges: Cleaning noisy live data in real time and tuning the LSTM architecture for accuracy without overfitting.

Outcome: The LSTM model outperformed traditional statistical baselines in forecasting accuracy across the evaluation period.
